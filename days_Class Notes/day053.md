*리소스 추가
1. pom.mxl
<build> <-여기 아래
		<resources>
			<resource>
				<directory>src/main/resources</directory>
			</resource>
		</resources>   

2. 프로젝트에 소스폴더 생성
- src/main/resources와 일치시켜야됨

3. root-context.xml 분산연결로 교체
```
 	<context:property-placeholder location="classpath:config/application.properties"/>
	<bean id="dataSource" class="com.zaxxer.hikari.HikariDataSource">
	<property name="driverClassName" value="${application.driverClass}"></property>
	<property name="jdbcUrl" value="${application.url}"></property>
	<property name="username" value="${application.username}"></property>
	<property name="password" value="${application.password}"></property>
	</bean>
```

4. properties 교체

application.driverClass=net.sf.log4jdbc.sql.jdbcapi.DriverSpy
application.url=jdbc:log4jdbc:oracle:thin:@localhost:1521:xe
