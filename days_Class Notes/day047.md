```

</select>

script : 
petSelect.addEventListener("change", function() {
  const petId = this.value;
  filterBrandsByPet(petId);
  filterFoodsByPetAndBrand(petId, null);
});

```
- "change" -> 변경사항 발생 시 함수 실행
- const petId = this.value; -> 상수에 option value값을 넣어줌(한 번만 할당 가능, 이후 변경 불가)
-  petId, null -> 첫 번째 인자, 두 번쨰 인자(미정)

```
function filterBrandsByPet(petId) {
  const options = brandSelect.options;
  for (let i = 0; i < options.length; i++) {
    const opt = options[i];
    if (!opt.dataset.pet) { opt.style.display = "block"; continue; }
    opt.style.display = (opt.dataset.pet === petId) ? "block" : "none";
  }
  brandSelect.value = "";
}
```
- const options = brandSelect.options; -> select 옵션 안에 있는 요소들 반복문에서 사용
- === : ==는 값만 비교하는 반변 ===는 타입도 비교하여 판단, 좀 더 명확해짐
        -> 반면 다르다는 표현의 경우 !=, !==로 =이 하나씩 적음 주의!
- display : "block" -> 블록 요소 형태로 보이게 하기(막는거 아님 주의)
- display : "none' -> 화면에서 숨김, 공간 차지x 클릭x
- visibilty : hidden -> 보이지 않지만 공간 유지
- brandSelect.value = ""; -> 값 초기화를 위해 삽입
- <option value="">-- 브랜드를 선택하세요 --</option> -> 그리고 이런 빈 value값을 주면 초기화 화면으로 보이게 됨

```
function filterFoodsByPetAndBrand(petId, brandId) {
  const options = foodSelect.options;
  for (let i = 0; i < options.length; i++) {
    const opt = options[i];
    if (!opt.dataset.brand) { opt.style.display = "block"; continue; }
    const matchesPet   = (opt.dataset.pet === petId);
    const matchesBrand = (!brandId || opt.dataset.brand === brandId);
    opt.style.display = (matchesPet && matchesBrand) ? "block" : "none";
  }
  foodSelect.value = "";
}
```
- function 함수명(조건1, 조건2) -> 조건 여러 개 설정 가능
- data- 속성 : 사용자 정의 속성 
- dataset : data- 속성을 접근할 때 사용, 객체화.
