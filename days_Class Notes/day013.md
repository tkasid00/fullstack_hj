# ■ Java

1. for(반복 횟수 알 때)
    - for (int i = 1; i <= 3; i++) {System.out.print(i + "\t");}

2. while(반복 횟수 모를 때-게시판) : 조건만 봄
    - int i = 1; ->초기문 위로 뺌.
	- while (i <= 3) {System.out.print(i + "\t");i++;}

		
3. do while(무조건 1번은 실행 필요)
	- i = 1;
	- do{System.out.print(i + "\t");i++;} while (i <= 3); ->일단 실행하고 while을 뒤로 뺌.
