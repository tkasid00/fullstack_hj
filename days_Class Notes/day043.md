1. MVC1 vs MVC2
 1) 구성 방식 ★
  - MVC1 : JSP가 MODEL, VIEW, CONTROLLER 역할을 모두 수행(스파게티 코드)
  - MVC2 :  MODEL(JAVA), VIEW(JSP), CONTROLLER(SERVLET) 분리

 2) 유지보수성 
  - MVC1 : 낮음(JSP에 로직이 포함되어 재사용 어려움) - 간단한 웹
  - MVC2 : 높음(MODEL, CONTROLLER 재사용) - 대규모
