```java
    1. 이 경우 main에서 for문을 이용해 출력 필요
    public void process_avg(Score std) {
		std.setAvg((std.getKor()+std.getEng()+std.getMath())/3);
	}
            ▼            ▼              ▼           ▼
    2. main에서 메서드만 돌리면 됨        
	public void process_avg(Score[] std) {
		for(Score s : std) {s.setAvg((s.getKor()+s.getEng()+s.getMath())/3);}
	}
```
