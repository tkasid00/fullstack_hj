*이미지 파일 관련 META-INF 내 context.xml 삽입, 
        기존 MANIFEST.MF는 삭제해도 무방
```
<?xml version="1.0" encoding="UTF-8"?>
<Context allowCasualMultipartParsing="true">
</Context>

```


*뷰 페이지 스크립트 이렇게도 작성 가능 - error가 비어 있다면
```
<c:if test="${not empty loginError}">
	<script>
		alert(${loginError});
	</script>
</c:if>
```
