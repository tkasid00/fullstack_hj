**jsp->th**

1. 컨트롤러에 매핑 걸기
2. 헤더 및 모든 경로들에 폴더경로 하나 끼우기
3. 조건식 변경
    1) if
        ```java

                        <!-- 이전 -->
                        <c:if test="${foodpaging.start > 10}">
                            <li class="page-item">
                                <a href="#" class="page-link"
                                onclick="foodList(${foodpaging.start - 1})">이전</a>
                            </li>
                        </c:if>

        ```


        ```java
                    <!-- 이전 -->
                    <li class="page-item"
                        th:if="${foodpaging.start > 10}">
                        
                        <a href="#" class="page-link"
                        th:onclick="|foodList(${foodpaging.start - 1})|">이전</a>

                    </li>

        ```


    2) for each
        ```java
                        <!-- 페이지 숫자 -->
                        <c:forEach var="i" begin="${foodpaging.start}" end="${foodpaging.end}">
                            <li class="page-item <c:if test='${i == foodpaging.current}'>active</c:if>'">
                                <a href="#" class="page-link"
                                onclick="foodList(${i})">${i}</a>
                            </li>
                        </c:forEach>


        ```


        ```java
                    
                    <!-- 페이지 숫자 -->
                    <li class="page-item"
                        th:each="i : ${#numbers.sequence(foodpaging.start, foodpaging.end)}"
                        th:classappend="${i == foodpaging.current} ? 'active'">
                        <a href="#" class="page-link"
                        th:onclick="|foodList(${i})|"
                        th:text="${i}">1</a>
                    </li>


        ```


       3) forEach 선택지 옵션

       ```java

       ``` 

       ```java
			<div class="col-4 mb-3">
			  <label class="form-label">브랜드 :</label>
			
			  <select name="brandid" class="form-control">
			    <option value="">-- 선택 --</option>
			
			    <option th:each="b : ${brandlist}"
			            th:value="${b.brandid}"
			            th:text="${b.brandname}">
			    </option>
			  </select>
			</div>
       ```

       **필터링 페이지 설계**
       1. DB : 기존 데이터 활용 - 신규 테이블 X
       2. 필터링 리스트
            [ 최소 필터 ]  → 강아지/고양이, 연령, 건식/습식, 브랜드, 사료이름 (사료 등록 재활용) 
            [ 권장 필터 ]  → 그레인프리, 원산지, 영양소라벨
            [ 검색어 ]     → 선택(기존 검색기능 활용)
            [ 고급 필터 ]  → 영양소 범위 입력, 칼로리, 세부 성분
