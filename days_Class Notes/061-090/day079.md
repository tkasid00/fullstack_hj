*자바스크립트 보충 공부

1. day03 도구 세트
- 배열 (array)
- forEach
- createElement 
- appendChild
- innerText

```java
 		<div>
	 		<button id="btn">출력</button>
			<ul id="list"></ul>
 		</div>
 		
 		<script>
 		document.addEventListener("DOMContentLoaded", function(){
 			const button = document.querySelector("#btn");
 			const list = document.querySelector("#list");
 			const pets=["고양이", "강아지", "토끼"];
 			
 			button.addEventListener("click", function(){
                list.innerText="";
 				
                pets.forEach(function(pet){
// 					console.log(pet);
				const li = document.createElement("li");
				li.innerText=pet;
				list.appendChild(li);
 					
 					
 				});
 			});
 		});
 		
 		</script>	
```
1) 구조
- const list = querySelector("#list") → 기존 ul 선택
- const li = createElement("li")      → 새 li 생성
- appendChild(li)                     → ul 내부에 붙임

2) list.innerText=""; 초기화 이유 : 누를 때마다 중복 출력 방지.
