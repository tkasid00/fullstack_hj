1. day01 도구 세트
- querySelector
- addEventListener
- value 
- innerText

2. 사용
```html
 		<div>
 			<input type="text" id="nameInput" placeholder="이름 입력">   
 			<button id="btn" >확인</button>
 			<div id="result"></div>
 		</div>
 		
 		
 		<script>
 		document.addEventListener("DOMContentLoaded", function(){
 			const button = 	document.querySelector("#btn");
 			const input = document.querySelector("#nameInput");
 			const result = document.querySelector("#result");
		
 	 		button.addEventListener("click", function(){
 	 			const name = input.value;
 	 			result.innerText = name + "님 환영합니다!";
 	 			
 	 			
 	 		});

 		});
 		
 		</script>
```
1) querySelector로 받아오는 id값이 const인 이유 : 참조하는 대상이 변하지 않음
2) value;와 value(); 
    - value; : 값 (property), 정보
    - value(); : 함수 실행 (method), 행동
3) input : 객체 안에 저장된 데이터 꺼내기
4) 흐름 : 이벤트와 기능 틀 잡기 - 참조할 아이디 설정 - 이벤트 걸기
5) DOM(Document Object Model) : HTML 문서를 JS가 조작할 수 있게 만든 객체 구조
6) DOMContentLoaded : DOM 트리 생성 끝났을 때 실행
7) HTML = 설계도  /  DOM = 실제 건물  /  JS  = 건물 안에서 가구 옮기는 사람



1. day02 도구 세트
- querySelector
- addEventListener
- value 
- if/else
- innerText

2. 사용
```html
 		<div>
			<select id="petSelect">
			<option value="">종 선택</option>
			<option value="1">고양이</option>
			<option value="2">강아지</option>
			</select>
			
			<div id="result"></div>
 		</div>
 		
 		
  		<script>
 		document.addEventListener("DOMContentLoaded", function(){
 			const select = document.querySelector("#petSelect");
 			const result = document.querySelector("#result");
 			
 			select.addEventListener("change",function(){
 				const pettype = select.value;	
 				if(pettype==="1"){
 					result.innerText="고양이를 선택했습니다";
 				}else if(pettype==="2"){
 					result.innerText="강아지를 선택했습니다";
 				}else{
 					result.innerText=""
 				}
 			});
 		});

 		</script> 
```
1) const pettype = select.value; 는 따로 빼는게 유리
2) select = 객체  /  select.value = 그 객체 안의 값   <-이 개념이 조금 헷갈림 어느 타이밍에 써야 할지
