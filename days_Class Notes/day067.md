*서비스단에서 파일+아이피 처리 방법
```
	@Override
	public int insertForUpload(MultipartFile file, SboardDto dto) {
		if(!file.isEmpty()) {	//파일이 존재하면
			String fileName = file.getOriginalFilename();	//원본 파일 이름 사용
			String uploadPath = "C:/file/";
			File img = new File(uploadPath+fileName);  //임포트는 import java.io.File; 선택
			try {
				
				file.transferTo(img);  //파일 올리기  - 트라이캐치
				dto.setBfile(fileName);
				
			} catch (Exception e) {
				e.printStackTrace();
			}
		}
		try {
			dto.setBip( InetAddress.getLocalHost().getHostAddress()); //아이피 
		} catch (UnknownHostException e) {
			e.printStackTrace();
		}
		return dao.insertForUpload(dto);
	}

```

```java
<script>
    function toggleContent(id) {
        const row = document.getElementById("content-" + id);
        row.style.display = (row.style.display === "table-row") ? "none" : "table-row";
    }
</script>
```
 1) toggleContent 
  - 상세 행 아이디 값을 다음과 같이 지정 : <tr id="content-${r.reviewid}">
  - const row로 상수값으로 받아와 이벤트 걸어줌
  - table-row 는 display 속성 중 테이블 형태로 보이게 랜더링
  - row.style.display = (행.스타일.보이는방식 === 보인다)? '닫기' : '보이기'
  - 클릭 기능은 다음과 같이 html에서 지정 : <tr class="review-row" onclick="toggleContent(${r.reviewid})"> 
    ->스크립트에서 지정하려면 일일히 개별 값을 넣어줘야됨 toggleContent(7) 이런 식으로 영원히...
