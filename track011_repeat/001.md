1. <script> 태그는 src 속성과 내부 코드를 동시에 가지지 못합니다.

```js
<script src="file.js">
  alert(1); // src 속성이 사용되었으므로 이 코드는 무시됩니다.
</script>
```

> 다음과 같이 수정하여 사용
```js
<script src="file.js"></script>
<script>
  alert(1);
</script>
```

2. 기본 사용법
1) alert 창 띄우기
```js
//기본
<script>
alert('테스트');
</script>
```

```js
//외부 스크립트 가져오기
<script src="alert.js"></script>
```

2) 형 변환
- alert는 기본적으로 문자열 -> String(value)를 통해 변환 가능
```js
let value = true;
alert(typeof value);  //boolean
value = String(value); //'true' 저장
alert(typeof value); //string
```

- 숫자형은 수학 관련 함수/표현식에서 자동형변환이 일어남
```js
alert( "6" / "2" ); //3
```
- 명시적 형변환 : Number(value)
```js
let str = "123";
alert(typeof str);
let num = Number(str);
alert(typeof num); 
```
