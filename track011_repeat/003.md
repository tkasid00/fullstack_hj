**B. 논리연산자**
1. OR (||)
    - 첫 번째 truthy 찾음
    ```js
    alert( 1 || 0 ); // 1 (1은 truthy)

    alert( null || 1 ); // 1 (1은 truthy)
    alert( null || 0 || 1 ); // 1 (1은 truthy)

    alert( undefined || null || 0 ); // 0 (모두 falsy, 마지막 값을 반환)


    true || alert("not printed");  // 첫 번째 값이 truthy -> 중지
    false || alert("printed");  // 메시지 출력
    ```
2. AND (&&)
    - 첫 번째 falsy 찾음
    ```js
    // 첫 번째 피연산자가 truthy -> 두 번째 피연산자를 반환 -> 중지
    alert( 1 && 0 ); // 0
    alert( 1 && 5 ); // 5
    alert( null && 5 ); // null
    alert( 0 && "아무거나 와도 상관없습니다." ); // 0
    alert( 1 && 2 && null && 3 ); // null
    alert( 1 && 2 && 3 ); // 모두 truthy일 경우 마지막 값 3 출력
    ```

**C. nullish 병합 연산자 '??'**
1. 기본 형태 : a ?? b 
    - x = (a !== null && a !== undefined) ? a : b;
    - a가 null도 아니고 undefined도 아니면 a 그 외의 경우는 b

2. ??와 ||의 차이
    - ??는 첫 번째 정의된(defined) 값을 반환
    - ||는 첫 번째 truthy 값을 반환
        ```js
        let height = 0;
        alert(height || 100); // 100
        alert(height ?? 100); // 0
        ```
3. 제약사항
    - ??는 &&나 ||와 함께 사용 불가

2. 과제

    1) 아래 코드 실행 시 마지막으로 출력되는 값
    ```js
    let i = 3;

    while (i) {
    alert( i-- );
    }
    ```
    - 답 : 1    <br/>
    - 해설 :    <br/>
        i-- 조건으로 인해 3-2-1 순서로 출력.  <br/>
        while(i) 는 boolean 판별값이기 때문에 0은 false로 변환됨. <br/>
        따라서 1까지 출력이 되고 0에서 반복문 중지

    2) while 예시의 출력값 비교
    ```js
     // 전위형 증가 연산자를 사용한 경우(++i)
    let i = 0;
    while (++i < 5) alert( i );
    
     // 후위형 증가 연산자를 사용한 경우(i++)
    let i = 0;
    while (i++ < 5) alert( i );
    ```
    - 답 : <br/>
        ++i -> 1-2-3-4    <br/>
        i++ -> 1-2-3-4-5  <br/>

    - 해설 :    <br/>
        ++i의 경우 i를 증가시킨 후 비교->출력의 순서를 거침 <br/>
        i++의 경우 비교 후 증가->출력의 순서를 거침     <br/>
        -> 연산자 우선순위상 ++이 >보다 우선순위가 높지만 <br/>
        i++은 i값 사용(비교)  후 증가됨

   3) for 예시의 출력값 비교
   ```js
     // 전위형 증가 연산자를 사용한 경우(++i)
    for (let i = 0; i < 5; ++i) alert( i );
    
     // 후위형 증가 연산자를 사용한 경우(i++)
     for (let i = 0; i < 5; i++) alert( i );
   ```     
    - 답 : 0-1-2-3-4로 동일 <br/>

    - 해설 :    <br/>
        alert( i )는 이미 조건을 통과한 현재값 출력 <br/>
        증감식은 반복문 끝에서 실행되기 때문에 alert( i )에 영향 없음. 

    4) for 반복문 사용하여 짝수 출력하기

    }
