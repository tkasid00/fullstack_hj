## 🐾 Pawject (v1~v4) | 반려동물 건강 & 사료 종합 플랫폼

> 게시판 기반 MVP에서 출발해  
> 검색/페이징/AJAX/업로드 → 외부 API 연동(OCR/ChatGPT/채널톡/지도·날씨) →  
> React 분리 아키텍처 + JWT + Redis + CI/CD 적용까지
