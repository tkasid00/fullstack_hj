🐾 from Stray to Home

유기묘 임시 보호 과정을 주제로 한 Java 콘솔 기반 시뮬레이션 프로젝트입니다.

보호자의 시점에서 고양이의 상태를 관리하며 돌봄 활동을 체험할 수 있습니다.

단일 사용자 제어문 구조에서 출발해 배열 기반 다중 계정, 인터페이스 중심의 객체지향 구조로 발전하며 구조화·확장성·상호작용을 강화해온 학습형 시리즈입니다.


🎯 주요 기능 요약

- 보호자 등록 및 로그인 시스템

- 고양이 상태 수치 관리와 돌봄 활동

- 환경 개선, 레벨 변화, 랜덤 이벤트 시스템


👉 각 버전의 세부 기능과 시연 영상은 [README 페이지] 에서 확인하실 수 있습니다.
