#1. 기본 세팅 : 컨트롤러에서 복사해서 가져옴, 메서드명 exec로 설정
```
public interface ReService {
		public void exec(HttpServletRequest request, HttpServletResponse response) 
				throws ServletException, IOException;
	}
```    

#2. 메서드별 설정

1. ReList(전체글보기)
 - Dao 불러오기 : ReDao dao = new ReDao();
 - 값 넘기기 : request.setAttribute("list", dao.selectAll());



2. ReInsert(글쓰기)
 - 넣을 값 설정하기 : dao의 insert 메서드와 일치 
            -> 형태 : int userid = Integer.parseInt(request.getParameter("userid")); 
 - 드커프리 : dao로 대체
 			-> ReDto dto = new ReDto(); , ReDao dao = new ReDao();           
 - 값 입력 : dto.setUserid(userid); 

 - 값 넘기기 : String result = String.valueOf(dao.insert(dto));
			  request.setAttribute("result", result);          


3. ReUpdateView(수정 페이지)
 - 받아올 값 지정 : dao에서는 reviewid로 특정하기로 설정함 
                    ->int reviewid=Integer.parseInt(request.getParameter("reviewid"));
 - 드커프리 : dao로 대체	
		    -> ReDao dao = new ReDao();
		       FoodList foodlist = new FoodList();
		        foodlist.exec(request, response);    //FoodList는 나중에 삽입한 값이라 여기서 추가로 들어가게 됨
 - 값 넘기기 : request.setAttribute("dto", dao.select(reviewid));
		      request.setAttribute("reviewid", reviewid);
              
4. ReUpdate(수정 실행)
 - 글꼴 깨짐 방지 : request.setCharacterEncoding("UTF-8");  
 - 넣을 값 설정하기 : dao의 update 메서드와 일치 
            -> int reviewid=Integer.parseInt(request.getParameter("reviewid")); 
 - 드커프리 : dao로 대체
 			-> ReDto dto = new ReDto(); , ReDao dao = new ReDao();           
 - 값 입력 : dto.setReviewid(reviewid);

 - 값 넘기기 : request.setAttribute("result", String.valueOf(dao.update(dto))); 

5. ReDeleteView(삭제 페이지) : ReUpdateView와 동일

6. ReDelete(삭제 실행) : ReUpdate와 동일 -> 값 넘길 때 메서드만 변경
